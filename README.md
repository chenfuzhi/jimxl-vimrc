# 我的vim配置文件

我的vim配置文件

## 初始化代码

``` shell
git submodule init
git submodule update
```

## 安装

rake -T 查看所有的任务

rake install 安装插件到用户目录下


## 一些功能热键

F2 打开文件目录
F3 taglist功能
F9 执行ctag命令
F4 切换头文件和cpp文件

ctrl + e 到行尾

ctrl + a 到行首

Tab 和 Shift Tab作为代码的缩进

gc 来切换注释
